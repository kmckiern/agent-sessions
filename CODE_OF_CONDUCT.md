# Code of Conduct

We want a welcoming, inclusive community. By participating in this project, you agree to:

- Be respectful and considerate in speech and actions.
- Assume good intent and give constructive feedback.
- Avoid harassment, discrimination, or abusive behavior.

## Scope

This Code of Conduct applies in project spaces (issues, PRs, discussions) and at any community events.

## Enforcement

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue or security advisory with as much context as you can share. Maintain confidentiality for all parties involved.

Project maintainers are responsible for clarifying standards and taking appropriate action.
